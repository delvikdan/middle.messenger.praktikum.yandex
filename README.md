# Приложение "Icewind Chat"

Одностраничное веб-приложение (SPA), реализованное на TypeScript c использованием собственного компонентного фреймворка.
Реализованы основные сценарии мессенджера: аутентификация, профиль пользователя, список чатов, сообщения, формы редактирования профиля и пароля.

## Стек и используемые технологии:

- TypeScript — строгая типизация, современный синтаксис.
- Собственный компонентный фреймворк — управление жизненным циклом и событиями компонентов (классы Block, EventBus).
- Handlebars — шаблонизатор для декларативного описания разметки компонентов.
- SCSS - препроцессор для работы со стилями. Стили компонуются отдельно по паттерну 7-1
- Mock Data — для эмуляции работы с бэкендом используются мок-данные.
- ESLint, Prettier, Stylint для поддержки качества кода.
- Проект собирается с помощью Vite.

## Возможности:

- Страницы входа и регистрации с валидацией инпутов.
- Список чатов, просмотр истории сообщений, отправка сообщения.
- Просмотр и редактирование профиля пользователя.
- Валидация всех форм отображается в console.log
- Система событий и управление DOM без сторонних фреймворков.
- Архитектура, пригодная для масштабирования.

## Установка

Установить зависимости проекта:

`npm install`

Собрать проект:

`npm run build`

Запустить проект на localhost:

`npm run start`

Запустить Eslint, Stylelint:

`npm run lint`

`npm run lint:style `
